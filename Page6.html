🎬 HTML Media

🌍 First Principle: The Web is More Than Text

Websites today are not just about static text and images. Modern webpages can directly include video and audio without relying on old plugins like Flash or QuickTime. HTML5 solved this with the <video> and <audio> tags.

🎥 Part 1: The Video Tag

The <video> tag is used to embed videos. At its simplest: 

<video src="my-video.mp4"></video>

But this just shows the first frame. To get proper controls (play, pause, volume, fullscreen), we add the controls attribute: 

<video src="my-video.mp4" controls></video>

✨ Useful Video Attributes

  • controls → Shows default player controls.
  • width / height → Set video size.
  • autoplay → Video starts automatically (usually requires muted).
  • loop → Replays automatically when finished.
  • poster → Thumbnail image before the video starts.

🎵 Part 2: The Audio Tag

The <audio> tag works the same way as video but without visuals: 

<audio src="theme-song.mp3" controls></audio>

It supports autoplay, loop, muted just like video.

🛠️ Part 3: Professional Embedding

Problem 1: Different browsers support different formats

Example: Chrome supports .webm, Safari prefers .mp4. The solution is to provide multiple <source> options inside <video>. 

<video controls poster="thumbnail.jpg">
  <source src="my-video.webm" type="video/webm">
  <source src="my-video.mp4" type="video/mp4">
  Sorry, your browser doesn't support videos.
</video>

Problem 2: Accessibility (Subtitles / Captions)

Use <track> to add subtitles or captions (stored in .vtt files). 

<video controls>
  <source src="video.mp4" type="video/mp4">
  <track src="captions_en.vtt" kind="captions" srclang="en" label="English">
  <track src="subtitles_es.vtt" kind="subtitles" srclang="es" label="Español">
</video>

Kinds of tracks: captions (for hearing impaired), subtitles (translation), descriptions (for visually impaired).

⚖️ Two Embedding Methods

1. Professional Method (with <source>)

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>
  • Pros: Reliable, supports all browsers, includes fallback text.
  • Cons: Requires multiple file formats.

2. Simple Method (with src attribute)

<video src="a.mp4" controls></video>
  • Pros: Very easy to write.
  • Cons: Works only if that one format is supported.

© Symbols & Entities

Sometimes we need to display special characters. We use HTML entities:

  • Copyright → &copy; → ©
  • Registered Trademark → &reg; → ®
  • Trademark → &trade; → ™
  • Less than → &lt; → <
  • Greater than → &gt; → >
  • Ampersand → &amp; → &
  • Double Quote → &quot; → "
  • Euro Sign → &euro; → €

Example footer usage:
<p>Copyright &copy; 2024 My Website</p>
